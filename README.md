## 数据库来源

### URL成语接龙

成语数据来源：[idiom-database](https://github.com/crazywhalecc/idiom-database)。该数据库数据来源于[中华新华字典项目](https://github.com/pwxcoo/chinese-xinhua)，并对其中新华字典中的成语部分进行了优化和修正：调整成语的拼音、增加了无声调的拼音和首尾字母的拼音，以方便编写成语接龙游戏。
